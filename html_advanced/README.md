# HTML Advanced 

## 📋 Project Overview

This project focuses on learning how to structure a web page using semantic **HTML5** tags.
No CSS or styling is applied — the goal is to understand proper HTML document structure and semantic markup.

---

## 🎯 Learning Objectives

* Understand **HTML5 guidelines and best practices**
* Create the skeleton of an **HTML5 page**
* Use **semantic HTML tags** to structure content
* Understand when to use **div vs span**
* Learn the semantic value of structural tags:

  * `header`
  * `main`
  * `footer`
  * `article`
  * `nav`
  * `section`
  * `aside`
* Implement proper heading hierarchy (`h1` → `h6`)
* Create different types of lists
* Understand image formats:

  * SVG
  * GIF
  * PNG
  * JPG
* Structure data in tables
* Embed video and audio
* Use iframes for external content

## 🧠 Key Concepts Practiced

### Semantic HTML

Using meaningful tags that describe the structure of the content rather than just presentation.

Examples:

* `<header>`
* `<nav>`
* `<main>`
* `<section>`
* `<article>`
* `<aside>`
* `<footer>`

---

### Content Structure

The project progressively builds a full HTML page by adding:

* headings
* paragraphs
* navigation
* lists
* quotes
* images
* media
* tables

---

### Styleguide

Multiple **styleguide pages** are used to demonstrate and test HTML elements such as:

* headings
* lists
* tables
* typography
* quotes
* media

---

## 🏁 Final Result

By the end of the project, a fully structured **semantic HTML webpage** is created that includes:

* navigation
* services
* works/portfolio
* about section
* blog/news
* testimonials
* contact section
* footer navigation
* media examples

All built using **pure HTML5 semantic structure** without CSS.
